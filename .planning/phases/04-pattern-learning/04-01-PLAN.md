Append two new exported functions to src/storage/events.ts (no changes to existing functions or table schema):

1. `getTopCravings(limit = 10)`: Queries all-time craving frequency.
   - SQL: `SELECT craving_id, craving_label, COUNT(*) as count FROM redirect_events GROUP BY craving_id ORDER BY count DESC LIMIT ?`
   - Return type: `{ craving_id: string; craving_label: string; count: number }[]`
   - Use db.getAllSync with `[limit]` param array

2. `getFeedbackStats()`: Per-craving yes/no/skip breakdown.
   - SQL: `SELECT craving_id, craving_label, SUM(CASE WHEN feedback = 'yes' THEN 1 ELSE 0 END) as yes, SUM(CASE WHEN feedback = 'no' THEN 1 ELSE 0 END) as no, SUM(CASE WHEN feedback = 'skip' THEN 1 ELSE 0 END) as skip, COUNT(*) as total FROM redirect_events GROUP BY craving_id ORDER BY yes DESC`
   - Return type: `{ craving_id: string; craving_label: string; yes: number; no: number; skip: number; total: number }[]`
   - Use db.getAllSync with no params (empty array `[]`)

TypeScript strict: declare explicit return types for both functions. Use `db.getAllSync<TheType>(sql, params)` pattern matching existing getHistory() usage.
