<task type="auto">
  <name>Task 1: Create theme module with useThemeColors hook</name>
  <files>src/theme.ts</files>
  <action>
Create `src/theme.ts`:

```ts
import { useColorScheme } from 'react-native';

export const darkColors = {
  background: '#0a0a0a',
  surface: '#1a1a1a',
  text: '#ffffff',
  textSecondary: '#cccccc',
  textMuted: '#666666',
  textSubtle: '#888888',
  accent: '#6366f1',
  destructive: '#ff6b6b',
  navText: '#444444',
};

export const lightColors = {
  background: '#f0f0f0',
  surface: '#e0e0e0',
  text: '#0a0a0a',
  textSecondary: '#333333',
  textMuted: '#888888',
  textSubtle: '#666666',
  accent: '#6366f1',
  destructive: '#cc3333',
  navText: '#888888',
};

export type ThemeColors = typeof darkColors;

export function useThemeColors(): ThemeColors {
  const scheme = useColorScheme();
  return scheme === 'light' ? lightColors : darkColors;
}
```

The hook defaults to dark (the existing design) when `useColorScheme()` returns `'dark'` or `null`.

<task type="auto">
  <name>Task 2: Apply theme to all screens</name>
  <files>app/index.tsx, app/result.tsx, app/history.tsx, app/stats.tsx, app/manage.tsx, app/add-mapping.tsx</files>
  <action>
For each screen, replace hardcoded color values with `colors.*` from `useThemeColors()`.

**Pattern for every screen:**
1. Add import: `import { useThemeColors } from '../src/theme';`
2. Inside the component function (before any returns), add: `const colors = useThemeColors();`
3. Replace hardcoded hex strings with the appropriate token:
   - `'#0a0a0a'` → `colors.background`
   - `'#1a1a1a'` → `colors.surface`
   - `'#ffffff'` → `colors.text`
   - `'#cccccc'` → `colors.textSecondary`
   - `'#666666'` → `colors.textMuted`
   - `'#888888'` → `colors.textSubtle`
   - `'#6366f1'` → `colors.accent`
   - `'#ff6b6b'` → `colors.destructive`
   - `'#444444'` → `colors.navText` (History/Stats/Manage nav links on home screen)
4. For `StyleSheet.create(...)` blocks: these are evaluated once outside the component, so they cannot use the dynamic `colors` value. **Migration approach:** convert stylesheet objects to inline styles where they reference colors, OR move `StyleSheet.create` inside the component after `const colors = useThemeColors()`. Prefer converting StyleSheet color entries to inline styles since the screens are not performance-critical.
5. `TextInput` in `add-mapping.tsx` needs `placeholderTextColor={colors.textMuted}` — add this prop.

**Important:** `useThemeColors()` is a hook — it must be called at the top level of the component function, before any conditional returns. (Per established "Hooks before guards" decision in STATE.md.)

After replacing colors, verify each screen compiles by running `npx tsc --noEmit`.
